shopping-cart-app
===================


## Installation
1. npm install.
2. bower install.

## Testing
run test with grunt test:unit

## For Development
1. grunt serve.
2. go to http://localhost:8000.

## To Build
run grunt build.

## To Serve dist files
1. grunt serve:dist.
2. go to http://localhost:9000.

## To run with node
1. node server.js.
2. go to http://localhost:5000.

demo is available [Here](http://shop-cart.azurewebsites.net).
